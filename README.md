# phonetic-password-tools
A free, privacy-focused Progressive Web App (PWA) to make passwords, codes, and alphanumeric text crystal clear when sharing aloud.

Features
Convert text to NATO phonetic alphabet
Generate strong passwords with instant phonetic versions
Speak aloud, copy, share, or QR code output
Supports 9 languages (English, French, German, Spanish, Italian, Dutch, Portuguese, Russian, Mandarin Chinese)
